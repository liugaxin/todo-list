# Todo-List 專案練習

## 功能說明

- 使用者可以新增一筆 todo 並指定名稱
- 使用者可以瀏覽全部 todo 清單，按名稱排序
- 使用者可以查看特定 todo 的詳細資料
- 使用者可以編輯一筆 todo 的名稱與完成狀態
- 使用者可以刪除一筆 todo

## 操作說明

###專案下載
```sh
git clone https://github.com/liugaxin/todo-list.git
```
###安裝package
```sh
npm install
```
###新增種子資料
```sh
npm run seed
```
###使用nodemon啟動伺服器
```sh
npm run dev
```
